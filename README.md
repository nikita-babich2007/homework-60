## Docker та Docker Compose (ДЗ 68)

Проєкт повністю контейнеризовано за допомогою Docker.

### Конфігурація

- **Dockerfile:** Створено образ на базі `node:lts`. Код копіюється у директорію `/app`, встановлюються залежності, відкривається порт `3000` та задається базова команда `CMD ["node", "src/app.js"]`.
- **Docker Compose:** Налаштовано два сервіси:
  1. `app` (Express-сервер)
  2. `mongo` (Локальна база даних MongoDB)
- **Зв'язок:** Сервіс `app` очікує запуску БД через `depends_on` та підключається до неї використовуючи змінну середовища `MONGODB_URI=mongodb://mongo:27017/homework`.
- **Volumes:** Налаштовано синхронізацію локальної директорії з `/app` у контейнері. Запуск сервера перевизначено командою `node --watch src/app.js`, що дозволяє редагувати код і бачити зміни миттєво без перезапуску контейнера.

### Як запустити

1. Переконайтеся, що у вас встановлено Docker та Docker Desktop.
2. Відкрийте термінал у корені проєкту та виконайте команду:
   ```bash
   docker-compose up --build
   ```
