# Homework-62

Цей проєкт — сервер на Node.js та Express.js за архітектурою MVC. В цьому оновленні додано проміжні обробники (middlewares) для безпеки, валідації та логування запитів, при цьому всі відповіді залишаються текстовими.

В проєкт інтегровано два шаблонізатори для серверного рендерингу сторінок (SSR):

- **PUG**: Використовується для маршрутів `/users` та `/users/:userId`. Шаблони генерують HTML зі списком користувачів та їхніми деталями.
- **EJS**: Використовується для маршрутів `/articles` та `/articles/:articleId`. Шаблони генерують HTML зі списком статей та їхнім контентом.
- Додано папку `public/css` з базовими стилями `style.css` для візуального оформлення сторінок.

## Встановлення та запуск

1. Склонуйте репозиторій на свій комп'ютер.
2. Перейдіть у папку проєкту та встановіть залежності:
   ```bash
   npm install
   # або
   yarn
   ```
