# Homework-63

Цей проєкт — сервер на Node.js та Express.js за архітектурою MVC.

## Cookies та JWT Авторизація

- **Статичні файли:** Налаштовано віддачу `favicon.ico` з папки `public`. Тег підключення додано у всі PUG та EJS шаблони.
- **Cookies (Налаштування):** Додано маршрут `POST /auth/theme`, який приймає тему користувача та зберігає її у cookies (використовується `cookie-parser`).
- **JWT (JSON Web Token):** \* Додано маршрути `POST /auth/register` та `POST /auth/login`. При успішній автентифікації генерується JWT.
  - Токен безпечно зберігається у cookies з прапорцем `httpOnly`.
  - Створено мідлвар `verifyJWT`, який захищає маршрути `/articles`, перевіряючи наявність та валідність токена з кук.

В проєкт інтегровано два шаблонізатори для серверного рендерингу сторінок (SSR):

- **PUG**: Використовується для маршрутів `/users` та `/users/:userId`. Шаблони генерують HTML зі списком користувачів та їхніми деталями.
- **EJS**: Використовується для маршрутів `/articles` та `/articles/:articleId`. Шаблони генерують HTML зі списком статей та їхнім контентом.
- Додано папку `public/css` з базовими стилями `style.css` для візуального оформлення сторінок.

## Встановлення та запуск

1. Склонуйте репозиторій на свій комп'ютер.
2. Перейдіть у папку проєкту та встановіть залежності:
   ```bash
   npm install
   # або
   yarn
   ```
