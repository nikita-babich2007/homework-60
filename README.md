# Homework-66

У цьому оновленні розширено інтеграцію з MongoDB. Реалізовано всі основні операції: створення, читання з проєкцією, оновлення (часткове та повне) та видалення як для поодиноких, так і для багатьох документів.

### 📖 Читання даних (Read)

- **Проєкція:** Метод `GET /articles` використовує `Article.find({}, 'title content')` для оптимізації запиту, повертаючи лише необхідні поля для відображення списку.

### ➕ Створення даних (Create)

- `POST /articles` (insertOne)
  - **Тіло запиту:** `{"title": "Нова стаття", "content": "Текст"}`
  - **Відповідь:** `Стаття "Нова стаття" успішно створена!`
- `POST /articles/bulk` (insertMany)
  - **Тіло запиту (масив):** `[{"title": "1", "content": "A"}, {"title": "2", "content": "B"}]`
  - **Відповідь:** `Успішно створено 2 статей!`

### 🔄 Оновлення даних (Update)

- `PATCH /articles/:articleId` (updateOne - часткове оновлення)
  - **Тіло запиту:** `{"title": "Оновлений заголовок"}`
  - **Відповідь:** `Оновлено документів: 1`
- `PUT /articles/:articleId` (replaceOne - повна заміна)
  - **Тіло запиту:** `{"title": "Нова", "content": "Повністю новий текст"}`
  - **Відповідь:** `Замінено документів: 1`
- `PATCH /articles/bulk` (updateMany)
  - **Тіло запиту:** `{"filter": {"title": "Стара"}, "update": {"title": "Нова"}}`
  - **Відповідь:** `Масово оновлено документів: N`

### ❌ Видалення даних (Delete)

- `DELETE /articles/:articleId` (deleteOne)
  - **Відповідь:** `Видалено документів: 1`
- `DELETE /articles/bulk` (deleteMany)
  - **Тіло запиту:** `{"filter": {"title": "Видалити мене"}}`
  - **Відповідь:** `Масово видалено документів: N`

## Встановлення та запуск
